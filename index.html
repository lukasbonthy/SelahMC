Connection Guide

Eagler (Browser)

  1. Launch the portal.
  2. Go to Multiplayer.
  3. Add a server.
  4. Paste the WSS address.
  5. Save and join.
wss://play.selahmc.net/

Java (Launcher)

  1. Open Minecraft Java Edition.
  2. Go to Multiplayer.
  3. Choose Add Server or Direct Connect.
  4. Paste the Java address.
  5. Save and join.
play.selahmc.net

Bedrock

  1. Go to Servers.
  2. Choose Add Server.
  3. Put in the Bedrock host.
  4. Enter the Bedrock port.
  5. Save and join.
bedrock.selahmc.net:19132
